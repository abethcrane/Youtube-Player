Youtube Question Player

Online classes are the big new thing, and everyone everywhere is uploading tutorial videos to youtube. But to make these properly effective there needs to be a way to keep user's attention and make sure they're not zoning out.

The Youtube Question Player hooks into a youtube video and allows you to set timepoints for questions to pop up. The user then has the opportunity to answer and check, and go back and re-watch that section if they get the wrong answer.

Multiple Question Tracks per Video

Any one video can have multiple question tracks, each displayed separately. This allows for different levels of testing and guidance.

Save and Submit

Tracks will have the option to generate a pdf of answers at the end of the video, and also to submit these to an instructor.
